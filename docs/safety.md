# Safety Notes

MyoBridge is an educational engineering project and is not a medical device.

Safety rules for body-connected testing:

- Use a commercial surface-EMG sensor.
- Use the ESP32 at 3.3 V.
- Use a USB isolator when connected to a laptop.
- Disconnect the laptop charger during body-connected tests.
- Do not connect electrodes across the chest, neck, head, or damaged skin.
- Do not change wiring while powered.
- Check for shorts before powering the sensor.
- Stop testing if any component heats up, smells unusual, or behaves unexpectedly.
