# Build Notes

## Milestone 1 — Simulated EMG pipeline

Created a simulated EMG-style signal-processing notebook. The notebook generates a synthetic EMG-like signal, plots the raw signal, rectifies it, calculates an RMS envelope, performs FFT analysis, windows the signal, and extracts basic features.

## Milestone 2 — ESP32 analogue acquisition

Built a basic ESP32 analogue acquisition pipeline using a potentiometer as a safe test input. The ESP32 streams timestamped ADC readings over serial, Python records the data to CSV, and a notebook plots the result.
